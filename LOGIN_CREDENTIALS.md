## ğŸ†˜ Troubleshooting

### "Invalid username or password"
- Double-check you're using the correct credentials
- Username: `employee` (lowercase, no spaces)
- Password: `employee123` (case-sensitive)

### "Cannot connect to server"
- Check if backend is running: `sudo supervisorctl status backend`
- Restart if needed: `sudo supervisorctl restart backend`

### "Page not loading"
- Check if frontend is running: `sudo supervisorctl status frontend`
- Restart if needed: `sudo supervisorctl restart frontend`

### Clear Browser Cache
If you see old data or errors:
1. Open browser DevTools (F12)
2. Right-click refresh button
3. Select "Empty Cache and Hard Reload"
